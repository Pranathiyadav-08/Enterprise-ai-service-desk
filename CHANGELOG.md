# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-07-19

### Added

- Enterprise service-desk chat interface and REST API.
- Intent-based agent routing for HR, IT support, company policy, and general queries.
- Retrieval-augmented responses backed by a local FAISS index and Ollama.
- Knowledge-base document upload, listing, deletion, and reindexing endpoints.
- Open-source project documentation and contribution guidance.
