# Future improvements

The following ideas are intentionally not part of the current implementation.

## Security and access

- Authentication with enterprise identity providers.
- Role-based access control for end users, administrators, and knowledge-base managers.
- Audit logs, document-level permissions, and configurable retention policies.

## Product experience

- Admin dashboard for operational visibility and knowledge-base health.
- Document upload portal with validation, metadata, and processing status.
- Speech input and output support.
- Voice-assistant interaction mode.
- Streaming model responses for faster perceived response time.

## Platform and scale

- Containerized deployment with Docker.
- Kubernetes deployment manifests and autoscaling.
- Redis for caching, rate limiting, and transient conversation state.
- PostgreSQL for users, configuration, audit events, and structured service data.
- Evaluation datasets, quality monitoring, and agent-routing analytics.
